# Code of Conduct

This project welcomes respectful collaboration.

## Expected behavior

- Be respectful and constructive
- Focus on technical discussion
- Help newcomers understand the project
- Give clear feedback with examples
- Assume good intent

## Unacceptable behavior

- Harassment or personal attacks
- Discriminatory language
- Spam or unrelated promotion
- Publishing private information

## Enforcement

Maintainers may remove comments, close issues or block participants who do not follow these guidelines.
